# Installing and configuring Julia and Pluto

- download and install "current stable release" of [Julia](https://julialang.org/downloads/)

Setting up Pluto:

1. start Julia
2. `] add Pluto`
3. back to main prompt: `use Pluto`
4. `Pluto.run()`
